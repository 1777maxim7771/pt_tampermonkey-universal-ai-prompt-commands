# Tampermonkey Universal AI Prompt Commands PT

Script Tampermonkey para substituir comandos curtos por prompts de IA prontos em chats de inteligência artificial.

O script substitui apenas comandos exatos por prompts preparados. O texto normal não é alterado. ChatGPT, Gemini, Claude, Copilot.

## Comandos

- `PT1` — Traduza integralmente e com precisão o texto fornecido para português.
- `PT2` — Resuma o texto fornecido em português conforme o sentido e o contexto.
- `PT3` — Crie em português um resumo temático curto da carta estritamente em uma linha.
- `PT4` — Traduza o texto fornecido para alemão simples e claro, nível A2-B1.
- `PT5` — Corrija o texto fornecido em português.
- `PT6` — Escreva uma resposta curta, educada e formal a esta carta em alemão simples, nível A2-B1.
- `PT7` — Explique em português simples o significado deste texto.
- `PT8` — Extraia todos os fatos importantes e organize-os em português: pessoas, organizações, endereços, datas, prazos, valores, documentos, requisitos, decisões, obrigações e próximos passos.
- `PT9` — Crie em português uma lista clara de ações necessárias com base no texto.
- `PT10` — Redija com base no texto uma carta oficial, educada e clara em alemão, nível A2-B1.

## Instalação

Instale o Tampermonkey, abra o link do userscript, confirme a instalação e digite o primeiro comando em um chat de IA.

[Userscript](https://raw.githubusercontent.com/1777maxim7771/pt_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js)

O script é instalado na extensão Tampermonkey, não no GitHub nem em um site. O GitHub apenas armazena o arquivo.

Tampermonkey → Dashboard → Utilities → Import from URL:

`https://raw.githubusercontent.com/1777maxim7771/pt_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js`
